## 📖 Overview <a name="about-project"></a>

It is a simple todo app that allows users to add and delete tasks. It also allows users to filter tasks by completed or uncompleted tasks. The project is built with React and uses React hooks to manage the state of the app. 

### Project Objectives <a name="project-objectives"></a>

- [x] users can add tasks.
- [x] users can delete tasks.

## 💻 Getting Started <a name="getting-started"></a>

- [Optional] Install git bash to your machine to enable you to clone this repo.
- install Visual Studio to be able to host a local live version.
- Install a browser to view the local live version.

To get a local copy up and running follow these simple example steps.
### Setup <a name="setup"></a>

- Open your GitHub account the repository's [link](https://github.com/yosaddis/react-todo)

### Prerequisites <a name="prerequisites"></a>

- Internet connection
- A github account
### Install <a name="install"></a>

- copy the repo's link and clone it by writing `git clone https://github.com/yosaddis/react-todo.git` on your git bash terminal.
- `npm install` to install the dependencies.

### Run tests <a name="run-tests"></a>

- You can check for errors by running linter tests found in the github flows.

### Deployment <a name="deployment"></a>

- run `npm run start` to run it locally.
